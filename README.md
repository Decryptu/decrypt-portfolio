# [decrypt.im](https://decrypt.im)

Personal portfolio and service pages built with [Next.js](https://nextjs.org/),
[Tailwind CSS](https://tailwindcss.com/), [Velite](https://velite.js.org/) and
deployed on [Vercel](https://vercel.com/).

![I am Designer logo](public/IamDesigner!English.png)

## Development

Create a `.env` file from `.env.example`, then run:

```sh
bun install
bun dev
```

## Structure

- `src/app/automations`: automation service page
- `src/app/ai-chat`: AI chat service page
- `src/app/projects`: portfolio projects
- `src/app/experiments`: visual experiments
- `src/lib/contact-actions.ts`: shared contact form action
