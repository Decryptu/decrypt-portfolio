## Movrank

Movrank is a no-code video creation tool I designed and developed end to end. It turns rankings, timelines, leaderboards, and spreadsheet data into animated MP4 videos for YouTube, TikTok, Reels, and presentations.

The product is built around a focused workflow: add data, choose a chart style, customize the video, and export. Users can upload a CSV, paste spreadsheet data from Excel or Google Sheets, or build a dataset by hand directly in the app.

## Core Product Focus

- **Bar chart races and line chart races**: two animated chart styles powered by the same rendering engine used for exports
- **Spreadsheet-to-video workflow**: CSV import, copy-paste support, and a manual table builder for users without a prepared file
- **Social-ready exports**: MP4 generation in landscape, square, and vertical formats for YouTube, TikTok, Reels, and presentations
- **Brand customization**: logos, avatars, custom colors, Pro accent colors, and watermark controls
- **Fast creation flow**: designed to go from raw data to a finished video in minutes without code or video editing software

## Key Features

- CSV and spreadsheet data import
- Manual dataset builder with a spreadsheet-style grid
- Bar chart race and line chart race previews
- MP4 export for multiple aspect ratios
- Custom colors, logos, avatars, and watermarks
- Free monthly video generation with paid upgrades for higher-quality exports
