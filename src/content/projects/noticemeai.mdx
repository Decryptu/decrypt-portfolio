## NoticeMeAI

NoticeMeAI is an AI SEO and LLM visibility tool I designed and developed end to end. It works like a Lighthouse-style scanner for AI search readiness, and now also checks how a website ranks inside ChatGPT, Claude, and Gemini for a specific prompt.

The product focuses on **LLM SEO**, **GEO**, **AEO**, and agent readiness. It checks the technical signals that matter for tools like ChatGPT, Claude, Perplexity, Gemini, Google AI Overviews, and emerging AI agents.

## Core Product Focus

- **AI visibility scoring**: grades websites across crawl access, extractability, structured data, citation readiness, prompt visibility, and agent readiness
- **LLM rank tracking**: runs one prompt across ChatGPT, Claude, and Gemini to see whether a site is cited, where it ranks, and which sources appear ahead of it
- **Lightweight scanning**: static HTTP fetches only, with no headless browser dependency
- **Secure by default**: SSRF-safe fetch layer, rate limits, scoped reports, and Redis-only report storage with a 24h TTL
- **Actionable output**: clear findings, Markdown/JSON exports, and AI-generated recommendations for paid scans

## Key Features

- Free homepage audit with core AI SEO checks
- Full paid scan with schema, well-known, agent-readiness, and commerce protocol checks
- Site-type-aware recommendations for SaaS, ecommerce, local business, documentation, marketplaces, and more
- Pro competitor comparison using lightweight homepage signal checks
- Live rank checks across ChatGPT, Claude, and Gemini from a single prompt
- Source comparison showing which cited pages outrank the target website
- Scan queue powered by BullMQ, Redis, and a separate worker process
- Stripe subscriptions with Starter and Pro plans
- Better Auth sign-in with Google and GitHub
- Markdown content negotiation and `/llms.txt` support for agent-friendly pages
